الأنواع الأساسية:


<ul> — unordered list: قائمة بنقاط (bullet).  <ul>   </ul>  غير مرقمه

<ol> — ordered list: قائمة مرقمة.    <ol>    </ol>

<dl> — description list: للـ "مصطلح — تعريف"  <dl>   </dl>  : <dt>  </dt>   -- <dd>  </dd> 
  
  
  <li> — كل عنصر جوه القايمة.    <li>  </li>
